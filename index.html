// 5 — Security
Passwords are stored as SHA-256 hashes — never plaintext. Changing a password here updates the hash; the old password is immediately invalid. Lockout applies after too many failed login attempts.
Gate for the dashboard entry screen
Gate for this terminal
Security note: SHA-256 hashing prevents plaintext exposure in source & storage. However, since this is a static app served over HTTPS (GitHub Pages), the file itself is the trust boundary — anyone who can serve a modified version of this file can bypass the gate. The real data protection is Google OAuth: the password gate is a UX lock, not a data lock. Your Google Sheet remains protected by Google's auth regardless.
